# ultr7a.com
WebXR game engine, blogging framework and static site generator.

Create VR environments from markdown :)


```bash
### Setup
cd ./client
npm install

cd ../server
go build

../generator
./build.sh
```

```bash
### Usage
cd ./client

# render markdown, compile front-end, and create static files to serve:
npm run generate

# dev mode
npm start
```
